0 Tổng Quan Công Nghệ & Cơ Sở Dữ Liệu

Ứng Dụng Di Động (Mobile Client)

  • Framework: .NET MAUI (C#) - Xây dựng ứng dụng Native trên đa nền tảng.
  • Bản đồ & Tọa độ: Kết hợp Native OS Geolocation API và Map Control.
  • Đa Phương tiện: Tính năng đọc văn bản tự động (Text-To-Speech) fallback.
  • Cơ sở dữ liệu: SQLite - Đóng vai trò là Offline Database. Gọi HTTP REST Get kéo Json Data từ Server về nạp vào bộ nhớ để sử dụng độc lập trên điện thoại.

Hệ Thống Quản Trị (CMS Web)

  • Web Framework: ASP.NET Core MVC
  • Truy xuất Data (ORM): Entity Framework Core (EF Core).
  • Cơ sở dữ liệu: PostgreSQL
  • Hệ sinh thái: Expose REST API JSON phục vụ định tuyến đồng bộ dữ liệu xuống Mobile.

Các Chức Năng Chính

  • Bản đồ & Định vị tự động: Giúp du khách và khách quốc tế tự tin khám phá các ngõ hẻm phố ẩm thực, không chệch lộ trình, thông qua la bàn dẫn đường và các điểm ghim ảo (POI).
  • Thuyết minh tự động: Tích hợp phát các đoạn âm thanh MP3 hoặc giọng đọc AI (Text-To-Speech) mô tả lại lịch sử, nguồn gốc các khu ăn uống, tạo chiều sâu văn hóa truyền tải thẳng vào tai nghe mà không cần đọc văn bản dài.
  • Cơ chế Ngoại tuyến (Offline Mode): Giải quyết triệt để nỗi đau mất mạng 4G/Wifi của khách quốc tế. Dữ liệu đã được API gắp sẵn vào SQLite local để mọi thao tác vuốt chọn thực đơn, nghe thuyết minh, xem vị trí đều tức thì, không bị cản trở bởi kết nối kém.
  • Đa ngôn ngữ: Cập nhật từ vựng đa ngôn ngữ.

3.1 & 3.2. Mobile App: Luồng Đăng Ký/Đăng Nhập Tài Khoản Người Dùng

sequenceDiagram
    actor Tourist
    participant TView as UI Views (Register/Login)
    participant DbSvc as DatabaseService
    participant SqLite as SQLite Local DB
    participant State as Application State

    %% Register Path
    Tourist->>TView: [ĐĂNG KÝ] Nhập Email, Name, Bấm Đăng ký
    TView->>DbSvc: RegisterUserAsync(...)
    DbSvc->>SqLite: Cố gắng INSERT INTO UserAccount
    alt Email Tồn Tại
        SqLite-->>DbSvc: SQLError (Constraint Unique)
        DbSvc-->>TView: Return Error
        TView-->>Tourist: Hiện Popup Cảnh báo
    else Đăng Ký OK
        SqLite-->>DbSvc: Success Rows = 1
        DbSvc-->>TView: Return True
        TView-->>Tourist: Báo Thành công, Dẫn về Đăng Nhập
    end
    
    %% Login Path
    Tourist->>TView: [ĐĂNG NHẬP] Nhập User, Pass
    TView->>DbSvc: GetUserAsync()
    DbSvc->>SqLite: SELECT * FROM UserAccount WHERE Auth
    SqLite-->>DbSvc: User Object / Null
    alt Hợp Lệ
        DbSvc-->>TView: User Account
        TView->>State: Ghi đè Global Session Cache
        TView-->>Tourist: Xóa Trang Auth, Rẽ sang Trang Chủ Shell
    else Sai Pass
        DbSvc-->>TView: Null
        TView-->>Tourist: Chớp đỏ, Yêu cầu thử lại
    end
